# 🏥 MediFix  
**Anjungan Pasien Mandiri & Sistem Antrian**

MediFix adalah aplikasi berbasis web yang dirancang untuk mendukung alur pelayanan fasilitas kesehatan secara **cepat, terintegrasi, dan efisien**, mulai dari **anjungan pasien mandiri**, **admisi**, **poliklinik**, hingga **farmasi**.

Aplikasi ini dikembangkan untuk membantu digitalisasi layanan kesehatan dengan fokus pada **antrian, integrasi BPJS**.

## 🚀 Fitur Aplikasi

### 🧾 Anjungan Pasien Mandiri
Digunakan langsung oleh pasien untuk mempercepat proses pelayanan:
- Ambil antrian admisi
- Pendaftaran poli + nomor antrian
- Cek kepesertaan BPJS
- Check-in pasien JKN
- FRISTA BPJS
- Pencarian data pasien
- SEP rujukan FKTP *(dalam pengembangan)*
- SEP poli *(dalam pengembangan)*
- Pembuatan Surat Kontrol Rawat Jalan *(dalam pengembangan)*
- Pembuatan Surat Kontrol Rawat Inap *(dalam pengembangan)*
